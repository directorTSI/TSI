Introduction: Loops in Python

Loops are fundamental programming constructs that allow statements to be executed repeatedly. They are essential for automating repetitive tasks and processing collections of data efficiently.

This lesson will cover the two main types of loops in Python: the `while` loop for indefinite iteration and the `for` loop for definite iteration over sequences, along with important control statements that alter loop behavior.

1. The `while` Loop: Indefinite Repetition

The `while` loop is used for indefinite iteration, meaning it continues to execute a block of code as long as a specified condition remains `True`. The exact number of times the loop will run is not known at the outset; it depends entirely on when its controlling condition becomes `False`.

Why Use `while` Loops?

You use `while` loops when you need to repeat an action until a specific condition is met, such as validating user input, polling for an event, or performing a task until a certain state is reached.

Syntax and Structure

The `while` keyword is followed by the condition and a colon. The loop's body is indented.

# Syntax for while loop
# while condition:
#     # Code to execute repeatedly
#     # (Must eventually change 'condition' to False!)

Loop Variable Initialization

It is essential to initialize variables controlling the loop's condition before the loop starts.

count = 0 # Initialization outside the loop
while count < 5:
    print(count)

Controlling the Loop

Inside the loop, you must modify variables that are part of the condition to eventually make it `False` and terminate the loop. Failure to do so results in an infinite loop.

g = 1
while g < 4: # Loop while g is less than 4
    print("Hello")
    print("Iteration: {}".format(g))
    g += 1 # Crucial update to prevent infinite loop

2. The `for` Loop: Definite Iteration

A `for` loop is used for iterating over a sequence (such as a list, tuple, dictionary, set, or string). This is known as definite iteration because the number of repetitions is fixed by the length of the sequence being iterated over.

Why Use `for` Loops?

You use `for` loops when you know exactly how many times you need to loop, or when you need to process every item in a collection.

Syntax and Flow

The `for` keyword is followed by a variable, the `in` keyword, the sequence, and a colon. The loop's body is indented.

# Syntax for for loop
# for variable in sequence:
#     # Code to execute for each item
#     ...
                            
# Output: This is a static example of syntax.

Execution Flow

The `for` loop processes each item in the sequence one by one until all items have been covered.

Flow of a `for` Loop

Start Loop

Are there more items in sequence?

No
Yes
End Loop
Assign next item to variable
Execute loop body
....loop back to check next item...

`for` Loop Examples

1. Iterating Over a List

The most common use is to perform an action for each element in a list.

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

3. Using the `range()` Function

To loop a specific number of times, use `range()` to generate a sequence of numbers (e.g., `range(5)` generates 0, 1, 2, 3, 4).

# Loop 5 times (i from 0 to 4)
for i in range(5):
    print("Loop number: {}".format(i))

4. Summing Numbers in a List

Loops are perfect for aggregating data, like calculating a sum or average.

numbers = [10, 20, 30, 40]
total = 0
for num in numbers:
    total = total + num
print("The sum is: {}".format(total))

3. Loop Control Statements

Python provides statements to alter the normal execution flow of loops. These are applicable to both `for` and `while` loops.

`break` Statement

The `break` statement immediately terminates the loop and transfers execution to the statement right after the loop.

# Find the first number divisible by 7
for number in range(1, 100):
    if number % 7 == 0:
        print("Found it! {}".format(number))
        break
print("Search complete.")
                            
Found it! 7
Search complete.

`continue` Statement

The `continue` statement skips the current iteration of the loop and moves immediately to the next iteration.

# Print only even numbers
for number in range(1, 11):
    if number % 2 != 0: # If number is odd
        continue        # Skip this iteration
    print(number)

`else` Clause in Loops

A loop's `else` clause runs if the loop completes normally (i.e., not terminated by a `break` statement).

# Search for a number (not found)
for n in [1, 2, 3]:
    if n == 4:
        print("Found 4!")
        break
else: # This 'else' belongs to the for loop
    print("Did not find 4.")
                            
Did not find 4.
